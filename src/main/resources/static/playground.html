召回率评估（Retrieval Eval）

基于标注的「查询 ↔ 相关片段」数据集，量化向量检索的召回质量。走管理员会话令牌鉴权，无需 API Key。
如何拿到真实 ID：在「问答 / 检索」选项卡选同一知识库，用与文档语义相近的查询发送（问答或仅检索均可），每个结果片段都会显示 chunk #主键ID 与 doc #ID。把目标片段的 chunk 主键填入 relevantChunkIds（按文档整体相关则填 relevantDocIds，对应 doc #）。
示例里的 12/13/20/7 只是格式占位，必须换成你库里的真实值，否则召回率恒为 0。
